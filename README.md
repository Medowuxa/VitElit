# VitElit

Static multi-page website for a pharmacy catalog. The project includes a home page, product catalog, sale page, company information and help page.

## Stack

- HTML5
- CSS
- SCSS source files

## Structure

```text
Vit/
  index.html      Home page
  katalog.html    Product catalog
  sale.html       Sale page
  info.html       Company information
  help.html       Help page
  css/            Compiled CSS and SCSS sources
  img/            Images and logo assets
```

## How to View

Open `Vit/index.html` in a browser, or run a static server from the repository root:

```bash
npx serve .
```

Then open the local address shown by the server and go to `/Vit/index.html`.

## Pages

- home page with popular products;
- catalog page with medicine cards;
- sale page;
- company information page;
- help and contacts page.
